# sql-superstore-analysis

This project analyzes the Superstore dataset using SQL.

## skills used:
- GROUP BY
- JOINS
- AGGREGATE FUNCTIONS
- WINDOW FUNCTIONS
- COMMON TABLE EXPRESSIONS (CTEs)

## Key Analyses
- Product analysis
- Regional performance
- Customer analysis
- Time based analysis
